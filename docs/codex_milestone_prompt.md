# Milestone Run Template

1. Read `AGENTS.md`, then `docs/codex_context.md`, then `docs/current_milestone.md`.
2. Create or update `docs/codex_milestone_result.md` with `STATUS: IN_PROGRESS`.
3. Inspect the smallest relevant file set and implement only the milestone.
4. Run focused validation first, then final aggregate checks once.
5. Finalize the result with `STATUS: COMPLETE`, `FAILED`, or `BLOCKED`, including evidence and remaining limitations.
6. Do not commit or push.
